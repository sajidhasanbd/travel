The Travel Booking Toolkit plugin works with the WP Travel Engine. It adds special widgets to the Travel Booking theme, making creating travel websites easier and smoother.

== Description ==

The Travel Booking Toolkit plugin is for the [Travel Booking theme](https://wptravelengine.com/wordpress-travel-themes/travel-booking/?utm_source=wp.org&utm_medium=travel_booking_toolkit&utm_campaign=wp_profile) by WP Travel Engine. It enhances the theme by adding extra features, including a Customizer and Widgets Section, to help you create professional travel and tour booking websites.

To make the most of this plugin, you must install and activate both the Travel Booking theme and the [WP Travel Engine plugin](https://wptravelengine.com/?utm_source=wp.org&utm_medium=travel_booking_toolkit&utm_campaign=wp_profile).

### WIDGETS FOR TRAVEL BOOKING WEBSITES

The Travel Booking Toolkit plugin includes a variety of widgets that you can use to add custom functionality to your travel booking website. These widgets include:

1. **WP Travel Engine: Call To Action Widget**
You can use the WP Travel Engine: Call To Action widget to create a custom call-to-action button for your website. This widget allows you to change the text, link, color, alignment and to open in a new tab.

2. **WP Travel Engine: Client Logo**
You can display multiple client logos on your website with the customizable WP Travel Engine: Client Logo widget. This widget lets you choose each logo's size, alignment, and border and add links to the client's websites.

3. **WP Travel Engine: Icon Text Widget**
You can spotlight key details on your travel booking site with the WP Travel Engine: Icon Text widget. You can add customizable text and icons, choose their color and alignment, and highlight special deals or popular places on your site.

4. **WP Travel Engine: Recent Posts**
You can showcase recent posts on your booking site with the customizable WP Travel Engine: Recent Posts widget, allowing you to select the number of posts, display thumbnails and post dates, and choose from three layouts.

5. **WP Travel Engine: Category List**
With the WP Travel Engine: Category List widget, you can display a list of trip categories on your website filtered by multiple criteria, such as destination, activities, trip types, difficulty, and trip tag.

6. **WP Travel Engine: Contact Widget**
The WP Travel Engine: Contact widget lets you add a contact form to your website. You can add a title, description, phone number, email address, and social icons. You can also change the order of the social icons.

7. **WP Travel Engine: Image Text Widget**
You can easily place images and text on your site using the WP Travel Engine: Image Text widget, making them look good and easy to read by picking the right size and alignment.

### WP TRAVEL ENGINE FEATURES

1. **Unlimited Travel Packages** — You can easily create and sell an unlimited number of tours. Effortlessly include tour details, gallery, itineraries, regular and sales prices, duration, trip facts, FAQs, and the included and excluded costs.

2. **Advanced Trip Search and Filter** —  You can use the existing search filters or create custom ones to enable your customers to locate the suitable trip according to destination, budget, duration, activities, trip types, and more.

3. **Smart Travel Booking Process** — The trip booking process is streamlined, fast, and easy, with automatic price adjustments based on the number of people selected and a booking form that only includes essential fields, resulting in faster bookings and more sales.

4. **Payment Processing** — WP Travel Engine accepts various payment gateways, allowing your customers to make bookings online and enabling you to receive payments promptly.

5. **Progress Bar** — WP Travel Engine features an attractive progress bar that visually displays the time of progress a guest has made in booking a tour. Studies have indicated that a progress bar can enhance bookings and sales.

6. **Enquiry Form** — WP Travel Engine has an inquiry form on the trip page, making it easy for customers to find, fill, and contact you, leading to more inquiries about your travel packages.

7. **Booking Email Notification** — You can personalize the message sent to your customers upon booking a trip. Additionally, whenever a traveler makes a booking on your site, you will be notified via email containing all the relevant booking details.

8. **Booking Details** — All booking information, including the travelers' details, is securely stored on your website. You can access any booking details or information with just a few clicks whenever needed.

9. **Discount Coupon** — You can generate various coupons to provide discounts to customers, encouraging them to make bookings.

10. **Create More Tabs** — You can use the existing four tabs (Overview, Itinerary, Costs, and FAQs) on each page and, if more are needed, easily create and reorder them!

11. **Unlimited Trip Facts** — You can create unlimited ‘Facts’ for trips, including group size, meals, accommodation, optimal visit times, and more, while adding them to various trips.

12. **Destination Page** — You can effortlessly establish pages for destinations like London, USA, and Nepal. This way, when a traveler visits a specific destination page, like Nepal, they can view all the trips associated with that destination.

13. **Activities Page** — You can develop pages for various activities like hiking, sightseeing tours, mountain biking, skiing, etc. It enables customers visiting your website to explore the diverse activities available and the trips related to them. For instance, customers who access a hiking activity page will be presented with all the hiking-related trips.

14. **Trip Type Page** — You can organize trips based on their specific types. For instance, you can plan trip categories such as dog-friendly, child-friendly, budget travel, nature walks, and weekend trips. So, when customers navigate to a specific type of page, like dog-friendly, they can view all dog-friendly trips.

15. **Currency** — You have the option to select the currency in which you want to sell your trip package.

WP Travel Engine is compatible with travel schema markup (JSON-LD format), aiding search engines in better understanding your website content. It transforms unstructured data into structured, enhancing the efficiency of search engine crawlers on your site and boosting organic visibility and ranking.

By meeting the criteria of the Schema Markup Validator Test and Rich Schema Test, WP Travel Engine provides your website with a competitive edge in SEO.

== Frequently Asked Questions ==

= WHAT FEATURES DOES THIS PLUGIN ADD? =
The Travel Booking Toolkit plugin is a collection of widgets that also adds extra options for customizing the Travel Booking theme's home page.

= WILL THE TRAVEL BOOKING TOOLKIT WORK ON OTHER THEMES? =
Initially, the plugin was designed to work with the Travel Booking theme by WP Travel Engine, but some widgets within the plugin might work with other themes.

= WHAT IS THE PLUGIN LICENSE? =
The plugin is open-source and licensed under GPL v2.0+, which means you can use, change, and distribute it freely if any changes you make are released under the same license.

= IS THE WP TRAVEL ENGINE PLUGIN DEPENDENT ON THE TRAVEL BOOKING TOOLKIT PLUGIN? =
No, the WP Travel Engine operates independently. Yet, using it with the Travel Booking theme and the Toolkit will enhance functionality and user experience by providing extra widgets and customization options for travel booking websites.

= HOW DOES THE WP TRAVEL ENGINE ENHANCE MY SITE'S SEO? =
WP Travel Engine support for travel schema markup in JSON-LD format. This schema markup enhances the understanding of search engines regarding your website content by converting unstructured data into a structured format. It allows for more efficient crawling by search engine bots, ultimately improving organic visibility and ranking.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select travel-booking-toolkit.zip from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download travel-booking-toolkit.zip
2. Extract the travel-booking-toolkit directory to your computer
3. Upload the travel-booking-toolkit directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
